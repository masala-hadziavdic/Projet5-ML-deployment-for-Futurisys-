# Projet 5 – ML Deployment for Futurisys

Déploiement d’un modèle de Machine Learning via une API REST développée avec FastAPI et déployée sur Hugging Face Spaces.

Projet réalisé dans le cadre de la formation **Data Scientist – Machine Learning (OpenClassrooms)**.

---

## Objectif du projet

Ce projet consiste à déployer un modèle de Machine Learning permettant de réaliser des prédictions via une API accessible en ligne.

L’objectif est de :
- entraîner et sauvegarder un modèle ML,
- exposer le modèle via une API REST,
- permettre des prédictions à distance,
- déployer l'application dans un environnement accessible publiquement.

---

## Technologies utilisées

- Python
- FastAPI
- XGBoost / Scikit-learn
- Pandas / NumPy
- Git & GitHub
- Hugging Face Spaces
